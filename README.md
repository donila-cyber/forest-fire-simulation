# • Моделирование лесных пожаров

Проект для математического моделирования распространения лесных пожаров.

## • Описание проекта

Разработка программного приложения для визуализации и анализа динамики лесных пожаров на основе системы дифференциальных уравнений.

## • Цели проекта

1. Создать математическую модель распространения пожара
2. Разработать графический интерфейс
3. Реализовать визуализацию в реальном времени
4. Добавить систему оповещений и анализа

## • Как использовать

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить: `python src/main.py`
